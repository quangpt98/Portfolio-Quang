# Portfolio - Fachinformatiker Anwendungsentwicklung

Herzlich willkommen auf dem Repository meines persönlichen Portfolios!

In diesem Projekt habe ich folgende Web-Technologien eingesetzt:
* **HTML5** - Strukturierung der Inhalte.
* **CSS3** - Styling mit Flexbox, CSS Grid und Responsive Design.
* **Font Awesome** - Integration von Icons für ein modernes UI mit CDN.
* **Google Fonts** - Typografie für bessere Lesbarkeit.
